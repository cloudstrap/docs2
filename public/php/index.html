Бързи инструкции

1. Създай приложението

app create myphpapp php-5.5 mysql-5.5

2. Промени кода

echo 'Hello Wolrd!' > index.php

3. Качи приложението

git push

4. Виж в браузър

http://myphpapp-<mydomain>.sapp.io/

Да, толкова е лесно!

Ако тази статия е неточна, остаряла или е изпусната важна информация, моля уведомите ни!
